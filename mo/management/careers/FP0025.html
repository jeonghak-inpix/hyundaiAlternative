Universe & Cloud Developer

[담당업무]

비즈니스에서 기검증 된 최신 AI기술을 바탕으로 Cloud SaaS 기반 Platform서비스의 설계/개발 업무를 수행합니다.

[주요업무]

  • Universe Backend Developer
    • 실시간/대용량 데이터 처리, ML 파이프라인 등이 포함된 AI Platform 서비스를 개발
  • Universe Frontend Developer
    • Universe의 당사/관리자/사용자 User Interface를 플랫폼 Component 서비스 형태로 구현
  • Cloud Developer & Architect
    • Cloud 기반 Big Data Platform(이하 BDP) 또는 Cloud Management Platform 설계/개발

[지원자격]

  • 학사 이상
  • Application 설계 및 개발 프로젝트 경험 3년 이상
  • Java, Spring, JavaScript, Python 활용 개발
  • Restful API 설계 및 구축 경험

[우대사항]

  • 데이터 모델링 및 ETL 프로세스 개발 경험 우대
  • Kubernetes 기반 서비스 개발 경험 우대
  • MLOps 관련 사용/구축 경험 우대
  • (Cloud) Cloud Infra 설계 및 관리 플랫폼 개발 경험 우대

[지원방법]

이메일 접수 recruiter@hyundaialt.com

서류 검토 > 실무 면접 > 임원 면접 > HR 면접 > 처우 협의 > 평판 조회 > 최종 합격(입사)

*상기 모집 절차는 모집 포지션 및 회사 사정에 의해 변경될 수 있습니다.

[기타사항]

  • 국내 취업 및 해외 출장에 결격 사유가 있는 경우 채용 불가 합니다.
  • 채용관련 부정 청탁 또는 입사 지원서 허위 사실 기재 시 채용 및 입사 취소됩니다.
  • 국가유공자 등 예우 및 지원에 관한 법률에 의거 취업지원 대상자를 우대합니다.
  • 전형 별 합격/불합격 대상자는 개별적으로 진행되며, 상세 일정은 전형 별 합격자에 한하여 개별 안내 드릴 예정입니다.
  • 공고 일정은 채용에 따라 변경, 연장, 조기 종료 될 수 있습니다.
  • 지원서를 포함한 채용 전형에서 제출한 내용이 사실과 다르거나 문서로 증빙이 불가할 경우, 혹은 고의적인 기재사항 누락이 확인될 경우 합격이 취소되거나 전형 상의 불이익을 받을 수 있습니다.
